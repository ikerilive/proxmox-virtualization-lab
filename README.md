## Overview
This repository documents my hands-on experience with **Proxmox Virtual Environment (VE)**.  
The lab covers VM & container management, backups, clustering, HA, migration, automation, and Proxmox Backup Server.

## Technologies Used
- Proxmox VE
- Linux KVM
- LXC Containers
- ZFS, NFS
- Ceph
- Proxmox Backup Server
- CLI tools (qm, pct, pvesh)
- Ansible
- Terraform

## What I Did
- Installed and configured Proxmox VE
- Created and managed Linux & Windows VMs
- Deployed and managed LXC containers
- Configured ZFS storage and NFS shares
- Implemented backups, snapshots, and restore operations
- Created VM and LXC templates
- Migrated VMs (including ESXi â†’ Proxmox)
- Used CLI, API, Ansible, and Terraform for automation
- Built a Proxmox cluster with HA and Ceph storage
- Deployed Proxmox Backup Server and verified backups
